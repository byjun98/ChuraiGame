## 슬라이드 23: 어려웠던 점

### 📌 화면 구성
- Challenge 리스트
- Before/After 수치

### 🎙️ 대사
> "개발하면서 어려웠던 점들입니다.  
> 
> **1. API 속도 문제**  
> RAWG API 매번 호출 → 9초 걸림  
> → DB 캐싱 도입 → **0.15초 (98% 단축)**  
> 
> **2. Cold Start 문제**  
> 신규 유저는 추천 불가  
> → 왓챠 스타일 온보딩 도입  
> → 3개만 평가해도 추천 가능  
> 
> **3. 저장 공간 문제**  
> N개 게임의 유사도 = N² 레코드  
> → 정규화 + Top-K 필터링  
> → **1억 개 → 50만 개 (99.5% 절약)**"
